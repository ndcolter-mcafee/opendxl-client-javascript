Class: Event

Event(destinationTopic)

Event messages are sent using the Client#sendEvent method of a client instance. Event messages are sent by one publisher and received by one or more recipients that are currently subscribed to the Message#destinationTopic associated with the event (otherwise known as one-to-many).
